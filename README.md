# PlayHub Manager

A game management platform built with modern web technologies.

## Technologies Used

- Vite
- TypeScript
- React
- shadcn-ui
- Tailwind CSS

## Getting Started

1. Clone the repository
2. Install dependencies:
```bash
npm install
```
3. Run the development server:
```bash
npm run dev
```

## Building for Production

To create a production build:
```bash
npm run build
```
